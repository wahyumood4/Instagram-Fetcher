# Instagram-Fetcher
Get direct url instagram media to array
```bash
npm i instagram-fetcher 
```
# Example
```javascript
const igm = require("instagram-fetcher");
var link = igm.download("https://www.instagram.com/p/CEvNcEMhkrT/?utm_source=ig_web_copy_link");
console.log(link);
```
